
# STRATEGYA

Sistema de Gestão Integrada desenvolvido pela ORBEEX para apoiar empresas na gestão estratégica, indicadores, planos de ação, riscos, oportunidades e conformidade com requisitos da ISO 9001.

## Tecnologias

- HTML
- CSS
- JavaScript
- Supabase
- Chart.js

## Principais Módulos

- Planejamento Estratégico
- Indicadores
- Planos de Ação
- Atas de Reunião
- Riscos e Oportunidades
